# python-ce-runtime

This project builds a [PortMaster][1] runtime for [pygame-ce][2] games.

Requeriments:

* Docker
* Docker compose
* make

Build the runtime:

    $ make build-runtime

An example game and a PortMaster compatible startup script is located at example folder.

[1]: https://portmaster.games/index.html
[2]: https://github.com/pygame-community/pygame-ce
